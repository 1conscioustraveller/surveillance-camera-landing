# Surveillance Camera Landing Page

A modern landing page for surveillance camera systems with dark/light theme support and interactive 3D elements.

## Features

- Dark/Light theme toggle
- Interactive 3D robot using Spline
- Neon purple styling
- Responsive design
- TypeScript for type safety
- Tailwind CSS for styling

## Project Structure

See [STRUCTURE.md](STRUCTURE.md) for detailed file structure.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

## Built With

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Spline 3D
